# Puro for Homebrew

This is an unofficial [Puro](https://puro.dev) tap for [homebrew](https://brew.sh).

## Initial setup

If you don't have homebrew, install it from their [homepage](https://brew.sh).

Then, add this tap:

```
brew tap Alberto97/puro
```

## Installing

To install the tool:

```
brew install puro
```
